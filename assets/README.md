## Updating Video URLs in README

Once you push this repository to GitHub, you should update the video URL in the main README.md:

Replace:
```
https://github.com/user/repo/assets/vortexshear_Ro0.1_Bu1.0_192x192.mp4
```

With your actual repository URL:
```
https://github.com/YOUR_USERNAME/YOUR_REPO_NAME/raw/main/assets/vortexshear_Ro0.1_Bu1.0_192x192.mp4
```

Or use the GitHub user content URL:
```
https://user-images.githubusercontent.com/YOUR_USER_ID/...
```

**Tip**: GitHub also supports uploading videos directly in the markdown editor by dragging and dropping, which automatically generates the correct URL.
